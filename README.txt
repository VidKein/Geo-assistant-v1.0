В этом проекте используется онлайн-карты - MapLibre(https://https://leafletjs.com/).
В этом проекте используется скрипт определения местоположения - Leaflet.Locate(https://github.com/domoritz/leaflet-locatecontrol).

Целью проекта является оптимизация работы при геодезичечких изысканиях.

V1.0
--------------------------------
Возможности:
- Создание базы точек для проведения геодезических работ.
    Которые сочтоят из : 
    1.Номер точки.
    2.Место участка работы (SOD-11...).
    3.Тип(в асфальте, в трубе(арматура), арматура, в стене, в заборе, ЕХТ).
    4.Координаты.
    5.Система координат.
    6.Высота.
    7.Дата внесния в базу. 
    ????8.Фото точки, для ориентации.
- Удобный єкспорт/имборт этих точек через офисные программы (через таблицы).
- Реализована функция динамического перевода из системы координат JTSK в WGS84(для размешения на карте).
--------------------------------
Отображение на карте(на мониторе/телефоне):
*СЛОИ:
    - Точки базовые и рабочии.
    - Информация по точкам:
        1.Номер.
        2.Координаты и высоты точек (отображаются при нажатии).
        3.Тип.
        4.Переключатель - измерено/неизмерено.
        ????5.Фото точки, для ориентации. С возможностью добовления на месте.
    - Карта:
        1.Базовая.
        2.Спутниковая.
*Календарь:Отображается календарь с отображениям (по дням) плана измерения(тахеометрия/нивелирования). 
*Условные обозначения:
    ТОЧКИ
    1.Базовые:
        а. нивелирование - зеленый треугольник;
        б. тахеометрические - синий перечеркнутый круг;
    2.Рабочие:
        а. нивелирование - зеленый круг с буквой внутри Н;
        б. тахеометрические - зеленый круг с буквой внутри Т;
    3.Маркер в виде галочки, обозначаюший что точка измерена или соорентирована.
    4.При включении геолокации. Точка местоположения - голубой круг (со стрелкой указываюшей напровления, зависит от телефона).
    5.Масштабная линейка - метрах/км в зависимости от масштаба.

????
*Экспорт/Импорт точек:
    1.Экспорт точек в различные форматы, с возможностью пересылки по почтеб месенжеру.
    2.Импорт точек, по штучно, списком.
*Настройки:
    1.Язык.
--------------------------------
Дополнительные функции:
- Отображение участков измерений (SOD-11...).
- Одображение на карте всех базовых точек (необходимых для работы).
- Одображение рабочик всех точек.
- Возможность отображения месторасположения устройства на карте, с целью ореинтации на рабочии точки.
